## usage

### set project dirs

use **;** split string

```
/path/to/your/dir1;/path/to/your/dir2;/path/to/your/dir3
```

![intro_2](./assets/vscode-manager-intro_2.png)

### search and open project

```
vsc-project
```

![intro_3](assets/vscode-manager-intro_3.png)
![intro_1](./assets/vscode-manager-intro_1.png)

### open new window

```
vsc-new-window
```
